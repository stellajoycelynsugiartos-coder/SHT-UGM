# SHT-UGM
Smart Health Tracker kami kembangkan atas dasar banyak masyarakat dan mahasiswa yang sulit mendapatkan layanan kesehatan gratis yang memadai
# Fitur Utama
**Input data:** user diminta untuk menginput data sesuai dengan kondisi saat itu.
**Kategorisasi dan analisis:** data yang telah diinputkan akan dikategorikan dan di analisis serta diberikan skor agar user dapat mengetahui skor kesehatannya saat ini.
**Saran dan Rekomendasi:** setelah data dianalisis dan skor akhir diketahui, program juga akan memberikan saran dan rekomendasi atas masalah kesehatan yang sesuai dengan hasil analisis dari program. Selain itu terdapat anjuran ke dokter bila skor kesehatan terlalu rendah agar user dapat memeriksakan diri lebih lanjut.
